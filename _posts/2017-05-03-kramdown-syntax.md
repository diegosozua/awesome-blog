The kramdown syntax is based on the Markdown syntax.

# Header

```
# Header 1
## Header 2
### Header 3
#### Header 4
```

# Formatted Text

This is a *italic* text.

```
*italic*
```

This is a **bold** text.

```
**bold**
```

This is a ***italic bold*** text.

```
***italic bold***
```
This is a 'I dont know'.

# Ordered List
```
 1. A
 2. B
 3. C
```
# Bullet List
```
  * 1
  * 2
  * 3
```

# Quotes

> This is a text in quote mode.

```
> This is a text in quote mode.
```
